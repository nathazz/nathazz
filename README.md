##  About Me

Software Developer with experience building systems in production. Focused on TypeScript/Node.js and Golang, with a strong background in REST APIs, external integrations, and automation across modern cloud infrastructure.

What I do:

- REST APIs and microservices with TypeScript (Node.js/Nest.js/Express) and Golang (Gin), with focus on reliability and clean architecture.
- AWS integrations (RDS, ECS, ECR, EC2, S3, Lambda, ELB, Fargate, CloudWatch, VPC), deployments with Docker + CI/CD pipelines, and messaging with RabbitMQ.
- Relational databases (PostgreSQL, MySQL, Oracle) and NoSQL (MongoDB, Redis), both design and optimization.
- Frontend with React.js and Next.js, delivering end-to-end features across the stack.
- Automation and scripting with Python and N8N.
- Testing (E2E, integration, unit) in agile environments.
- Also experienced with Java 8/17/21 and Spring Boot, bridging backend contexts when needed.

I enjoy working in collaborative teams, building useful things, and learning through real-world projects. I don't stick to a single stack, I believe there's always something valuable to learn from different technologies and domains.
Working on Linux daily and comfortable navigating infrastructure, code, and everything in between.
Open to remote Full Stack or Front-end/Backend opportunities.
